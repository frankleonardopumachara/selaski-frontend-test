# Selaski frontend test

Pasos para ejecutar el proyecto

1. Clonar el respositorio
2. Instalanar las dependencias usando el comando `npm install`
3. Ejecutar la app `ng serve`


El proyecto consta de una pantalla `Home` donde se visualiza 2 botones:
1. El boton `Usuarios` muestra un formulario donde se puede crear usuarios y una lista de usuarios registrados.
2. El boton `Ordenes` muestra una interfaz para crear ordenes y una lista de ordenes registradas.
